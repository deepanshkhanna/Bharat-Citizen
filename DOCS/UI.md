## One-Line Description

An AI-powered civic operating system that helps citizens discover government services, understand eligibility, gather required documents, report public issues, and track outcomes through guided workflows.

---

# PRODUCT PHILOSOPHY

Smart Bharat is NOT:

- A chatbot
- A government website
- A scheme directory
- A dashboard
- An AI startup product

Smart Bharat IS:

- A civic operating system
- A citizen-first platform
- A guided experience
- A workflow engine
- A trusted civic assistant

---

# CORE USER GOALS

Users come to Smart Bharat to:

1. Discover schemes
2. Understand eligibility
3. Gather documents
4. Report issues
5. Track complaints
6. Receive civic guidance
7. Access information in a simple way
